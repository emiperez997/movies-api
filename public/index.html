La mejor API de Películas

Si necesitas una API de películas para practicar Frontend, estas en el lugar correcto.

Uso

  • En cada petición es necesario utilizar la API key que recibes al momento de registrar tu mail.
    Se utilizaría de esta forma:

    /api/movies?apikey=YOUR_API_KEY

  • Obtener una película por id:

    /api/movies/:id

  • Buscar películas por titulo:

    /api/movies/search?title=TITLE

Registro

Para poder utilizar nuestra API es necesario registrarte con tu mail. Una vez enviado, te llegara la API key para usarla.
